## ✂️ Editing Instructions

### Software
- **Recommended:** DaVinci Resolve (free), Adobe Premiere Pro, CapCut

### Timeline (5-min video)
| Timestamp | Clip | Transition | Notes |
|-----------|------|------------|-------|
| 00:00–00:20 | Title + Intro | Fade in | Add background music |
| 00:20–00:45 | Problem statement | Cross dissolve | Use statistics overlay |
| 00:45–01:00 | Solution overview | Cross dissolve | Module icons animate in |
| 01:00–04:00 | Live demo recording | None (clean cuts) | Speed up waiting states |
| 04:00–04:30 | Architecture diagram | Zoom in to slide | Highlight tech stack |
| 04:30–04:55 | Closing + Thank you | Fade out | Contact/QR code overlay |

### Effects
- **Loading states:** Speed up 2x or cut completely
- **Button clicks:** Add subtle click sound effect
- **Score animations:** Keep at normal speed (visual impact)
- **Text overlays:** Bottom-third for feature names
- **Subtitles:** Enable for accessibility

---

> **Pro tip:** Practice each demo at least 3 times before recording. The 3-minute version should feel effortless — memorize the opening hook and closing line.
